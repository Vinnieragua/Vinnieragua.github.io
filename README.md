## Deployment

This website is designed for GitHub Pages deployment:

1. Push to GitHub repository
2. Enable GitHub Pages in repository settings
3. Select main branch as source
4. Site will be available at `https://username.github.io/repository`

## Content Management

Content is managed by directly editing HTML files:

- **Gallery**: Add video embeds and image links in `gallery.html`
- **Events**: Update performance dates in `events.html`
- **Blog**: Add new project posts in `blog.html`
- **Contact**: Update email and social media links in `contact.html`

## Customization

### Colors
Edit `css/style.css` to modify the 90s color scheme:
- Background: `#000000` (black)
- Primary text: `#FFFF00` (yellow)
- Accent: `#FF8C00` (orange)
- Links: `#00FFFF` (cyan)

### Effects
JavaScript effects are in the `js/` folder:
- `glitter.js`: Mouse trailing effects
- `counter.js`: Hit counter animation
- `guestbook.js`: Guestbook functionality
